# PRD: ブラウザ完結型メディアコンバーターシステム

## 1. 製品定義 (Product Definition)

### 製品目標 (Product Goals)
1.  **プライバシー優先の処理**: 100%クライアントサイド（ブラウザ内）でのメディア変換を実現し、データの外部流出を完全に防ぐ。
2.  **高性能なユーザー体験**: `FFmpeg.wasm` を活用し、大容量ファイル（動画・音声）でもブラウザ上で軽快に動作するドラッグ＆ドロップ対応のインターフェースを提供する。
3.  **拡張性と収益性の両立**: 将来の機能追加を見据えたサイドバーレイアウトを採用し、Amazonアソシエイト等の広告枠を自然に組み込む。

### ユーザーストーリー (User Stories)
- **コンテンツクリエイターとして**: PNG画像をWebP形式にブラウザ内で変換したい。ファイルを外部サーバーにアップロードすることなく、サイト高速化のための画像最適化を行いたい。
- **SNSユーザーとして**: MP4動画を別の形式に変換したり、サイズを圧縮したりしたい。手軽かつセキュアに共有用ファイルを作成したい。
- **プライバシーを重視するユーザーとして**: 私的な音声ファイルを変換したい。サービス提供者にデータを保存・解析される心配なく、ローカル環境で処理を完結させたい。
- **サイト運営者として**: 動画変換の待機中に、関連する商品のおすすめ（広告）を表示し、ユーザーに役立つツールやハードウェアを提案したい。

### 競合分析 (Competitive Analysis)
| 競合サービス | 長所 (Pros) | 短所 (Cons) |
| :--- | :--- | :--- |
| **CloudConvert** | 対応形式が非常に豊富、高品質。 | ファイルのアップロードが必要（プライバシーリスク）、大容量は有料。 |
| **Online-Convert** | 使いやすい、広範なサポート。 | 広告が非常に多い、アップロードが必要。 |
| **FFmpeg.wasm Demo** | 100%クライアントサイド、無料。 | UIが最小限で一般向けではない、一括処理などが不足。 |
| **Squoosh.app** | 画像変換・最適化に特化し非常に高性能。 | 画像のみ。動画や音声には非対応。 |

## 2. 技術仕様 (Technical Specifications)

### 技術要件分析 (Requirements Analysis)
- **フレームワーク**: Next.js (App Router)
- **フロントエンドロジック**: `FFmpeg.wasm` (v0.12+) を使用。`SharedArrayBuffer` を利用するため、`COOP/COEP` ヘッダーの設定が必須。
- **画像処理エンジン**: ブラウザの Canvas API を使用した高速処理。
- **デザイン系**: Tailwind CSS を使用し、「Flat & Simple」（Coffee Inc スタイル）なダークモードUIを実現。
- **パッケージ管理**: npm。

### 要件プール (Requirements Pool)
| ID | 要件内容 | 優先度 |
| :--- | :--- | :--- |
| **P0-1** | Canvas API による画像変換 (PNG/JPG/WebP) とリサイズ機能。 | P0 (必須) |
| **P0-2** | FFmpeg.wasm による動画変換・簡易圧縮機能。 | P0 (必須) |
| **P0-3** | Next.js での COOP/COEP ヘッダー設定。 | P0 (必須) |
| **P0-4** | ドラッグ＆ドロップ対応のファイルアップローダー。 | P0 (必須) |
| **P1-1** | 音声形式の変換 (MP3/WAV/AAC)。 | P1 (推奨) |
| **P1-2** | サイドバーレイアウト（メニュー切り替え：画像/動画/音声）。 | P1 (推奨) |
| **P1-3** | FFmpeg 処理の進捗表示（プログレスバー）。 | P1 (必須) |
| **P2-1** | Amazonアソシエイト用広告枠の配置。 | P2 (任意) |

### UI設計案 (UI Design Draft)
- **サイドバー (左側固定, 250px)**:
    - **上部**: ロゴとタイトル ("Media Converter")
    - **ナビゲーション**: Lucideアイコン付きメニュー（画像変換、動画変換、音声変換）
    - **下部**: Amazonアソシエイト用バナーエリア
- **メインエリア (右側)**:
    - **ヘッダー**: 現在選択中のメニュー名
    - **アップロード領域**: ドラッグ＆ドロップエリア
    - **設定エリア**: 変換先フォーマット選択、圧縮率スライダー、リサイズ設定
    - **結果表示**: ダウンロードリンク、プレビュー、関連商品リンク（広告枠）

### 未解決事項 / 確認ポイント (Open Questions)
1.  **FFmpegコアの選択**: `@ffmpeg/ffmpeg` v0.12 (マルチスレッド対応) をデフォルトとします。
